# 🎮 CareerQuest Game

CareerQuest is a real-time multiplayer coding battle platform where users compete by solving coding challenges in a game-like environment.

---

## 🚀 Features (v1.0)

- ⚔️ Real-time multiplayer battle using Socket.IO  
- 💻 Code Editor Battle (JavaScript execution)  
- 🧠 Dynamic question system with test cases  
- 🏆 Leaderboard system based on XP  
- 🎯 Matchmaking system (queue-based pairing)  
- 🎨 Game-style UI with animations  

---

## 🛠️ Tech Stack

- Frontend: React.js, Tailwind CSS  
- Backend: Node.js, Express.js  
- Realtime: Socket.IO  
- Code Execution: Child Process (Node sandbox)
